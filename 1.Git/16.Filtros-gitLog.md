# Adicionando filtros ao git log

Este argumento filtra os commits para mostrar apenas aqueles que foram feitos **desde** a data especificada.
> git log --since="Jan 1 2025" 

Filtra os commits para mostrar apenas aqueles que foram feitos **até** a data especificada.
> git log --until="Jan 1 2025"

É possível mesclar os dois para mostrar **entre** as datas:
> git log --since="Jan 1 2025" --until="Jan 5 2025"



Filtrar pelo autor:
> git log --author="Ana Francisca"



O log de commits simplificado:
> git shotlog 

Só a quantidade de commits e o autor:
> git shortlog -sn



Quantidades específicas de commits para ver:
> git log -3 //ele vai mostrar os 3 últimos commits feitos
