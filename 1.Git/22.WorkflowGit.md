# Workflows do Git

> Workflow = Fluxo de trabalho

Principais workflows do git:

- Contralized workflow;
- Feature branch workflow;
- Gitflow workflow.

## Centralized workflow 

- Branch único;
- Fácil transição para quem vem do SVN;
- Melhor para equipes pequenas.

## Feature branch workflow 

- Um branch par acada feature;
- Evita conflitos e permite trabalho paralelo;
- Funciona bem com equipes maiores.

# Gitflow
