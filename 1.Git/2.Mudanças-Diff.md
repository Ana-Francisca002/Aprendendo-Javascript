# Fazendo mudanças usando diff

> Commits atômicos: mudanças feitas em um só ponto, para não comprometer quebrar todo o projeto.

**Contexto do curso:** A única mudança feita foi no primeiro título da página, onde o William mostra como ver as mudanças e feitas e como podemos voltar para o estado anterior e tals. Depois de fazer essa mudança no título nós adicionamos a mudança para usar o git diff e depois commitar.

**Boas práticas:**
- As mensagens enviadas em inglês;
- Verbos no imperativo (add, change, remove, etc);
- A mensagem (obviamente) deve estar diretamente ligada a mudança que você fez.
