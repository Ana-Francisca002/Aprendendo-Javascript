# Trazendo commits para outras branchs

Caso você esteja trabalhando em um projeto com muitas branchs e quer trazer o algum commit de uma branch para outra sem precisar fazer um merge ou rebase, é possível usar o **cherry pick**. O comando:
- git cherry-pick hashdocommit 

O cherry pick serve para quando há muitos commits já feitos mas você só precisa de um ou 2 em específico para trazer par a branch que está trabalhando.
