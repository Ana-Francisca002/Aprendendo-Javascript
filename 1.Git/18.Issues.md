# O que são issues?

Se trata de uma ferramenta no github que serve para acompanhar e dar seguimento ao projeto, parecendo um fórum, onde problemas, sugestões, soluções, etc, são relatados com os outros colaboradores do projeto.

## Como criar uma issue?

1- Você pode criar manualmente;
2- Selecionar um trecho do código e criar a issue encima dele.

## Template de issue

Basta criar um arquivo **.github** e criar um arquivo **issue_template.md"**, no momento que você for criar a issue esse template vai surgir.

## Referenciando e fechando issue

Nos Github:
- Para **referenciar** uma issue basta ir nos comentário e colocar um **#** que o github vai te mostrar uma lista das issues.

No seu projeto local:
- Para **fechar** uma issue basta que no momento de fazer o commit você coloque alguma das "palavras chaves" para isso, uma dessas palavras seria **closes**. Um exemplo:
> Bla bla bla - Closes #2
A #2 indica qual issue vai ser fechada.
