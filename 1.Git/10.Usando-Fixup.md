# Juntando commit com fixup

Usando o seguinte comando:
> git rebase --fixup hashdocommit
Você vai corrigir o commit adicionado, o fixup cria um commit só para indicar que foram feitas as correções.

Com tantos commiit criados, podemos usar um parâmetro interessante para juntar todos esses commits:
> git rebase -i --autosquash hashdocommitantesdosfixup

Quando for trabalhar com feature branch muitas vezes é necessário fazer pequenas alterações nos arquivos, então, para não criar commits sujos se utiliza do fixup e squash.
