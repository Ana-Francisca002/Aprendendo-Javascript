# Juntando commits com squash

Caso você tenha dois commits diferentes e quer juntar eles você pode usar o squash, que é uma opção de parâmetro para o comando rebase.
> git rebase -i HEAD~2(esse 2 seria o número de commits que você quer juntar, lembrando que essa junção ocorre do seu commit atual)

Quando rodar o comando, você vai preciasar digitar, literalmente, squash a frente do commit que você quer que se junte com o outro.
