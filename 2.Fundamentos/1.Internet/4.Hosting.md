# O que é hospedagem?

Primeiramente vamos entender a diferença entre: página web, site, servidor, serviço web e mecanismo de busca.

- **Página web:** Documentos escritos em HTML que são exibidos pelo navegador;
- **Site:** Um conjunto de páginas web vinculadas e que compartilham um nome de domínio único;
- **Servidor web:** Um computador que hospeda um ou mai sites. "Hospedar" quer dizer que todos o site e seus arquivos estão disponíveis a partir desse computador;
- **Mecanismo de busca:** É um website que ajuda pessoas a encontrarem páginas web de outros websites;
- **Navegador:** Aplicação de software que recupera e renderiza páginas web.



# Servidor web

Primeiramente, vamos entender a diferença entre o hardware e o sorftware do servidor web:

- **Hardware:** É a máquina em si, que guarda os arquivos do website (html, css, js);
- **Software:** Essa parte é referente aos componentes que controlam como os usuários acessam os arquivos do site (servidor http para que o navegador possa visualizar a página web).

![Web server](/2.Fundamentos/1.Internet/img/web-server.svg)

Na comunicação entre cliente e servidor, o servidor tem os arquivos e o cliente quer acessar esses arquivos, o protocolo é responsável por retornar com o site ou dar mensagem de erro ao cliente. É isso aí.

## Diferença entre conteúdo estático e dinâmico

Um servidor web estático consiste em um computador (hardware) com um servidor HTTP (software). É chamado "estático" porque o servidor envia seus arquivos tal como foram criados e armazenados (hospedados) ao navegador.
**EX:** Portfólios Pessoais: Sites que exibem projetos ou obras de um artista, sem necessidade de interação, como um currículo online.

Um servidor web dinâmico consiste em um servidor web estático com software adicional, mais comumente um servidor de aplicações (application server) e um banco de dados (database). É chamado "dinâmico" porque o servidor de aplicações atualiza os arquivos hospedados antes de enviá-los ao navegador através do servidor HTTP.
**EX:** Blogs: Plataformas como WordPress, onde os posts são gerados dinamicamente a partir de um banco de dados, permitindo que visitantes comentem, visualizem diferentes postagens, e isso ocorre em tempo real.
