Flex

O display flex é uma versão melhorada do float, onde você tem um controle melhor dos itens em um container.


    Para o container

  • flex-direction: define o eixo principal;
  • justify-content: usado para organizar os itens no eixo row;
  • align-items: usado para organizar os itens no eixo column (alinha os itens);
  • align-content: usado para organizar os itens no eixo column (alinha o conteúdo)
  • flex-wrap: quebra a linha com os itens no container;
  • flex-flow: junção do flex-direction e flex-wrap.

    • Para o item

  • flex-grow: faz com que os itens se acomodem na div (distribui o espaço);
  • flex-basis: dá o tamanho para o elemento;
  • flex-shrink: permite ou não que o elemento diminua;
  • align-self: alinha um item em especifico (usando o :nth-child)
