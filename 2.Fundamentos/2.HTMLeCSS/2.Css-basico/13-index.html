Grid

  • grid-template-columns: define as configurações das colunas;
  • grid-template-rows: para definir as linhas da grid;
  • fr: fragmento, é uma medida interessante de distribuir as divs pela grid;
  • repeat: vai repetir tantas vezes alguma coisa ( repeat(10, 1fr) );
  • grid-column(row) span 2: a minha coluna ou linha vai fazer um span.

É possível usar o repeat para deixar o design da grid responsivo: repeat(auto-fill, minmax(320px, 1fr)). O auto-fill faz a grid se ajustar automaticamente e o minmax define a largura mínima e máxima.
